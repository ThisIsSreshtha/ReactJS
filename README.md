# ReactJS

A small React project scaffolded with Vite. This repository contains a simple React app (Vite + React + basic ESLint) used for learning and experimentation.

Repository: https://github.com/ThisIsSreshtha/ReactJS

## What's included

- Vite dev server with HMR
- React components under `src/`
- Basic ESLint configuration

## Requirements

- Node.js 16+ (recommended)
- npm (or yarn / pnpm)

## Quick start

1. Install dependencies

```powershell
npm install
```

2. Start development server

```powershell
npm run dev
```

3. Build for production

```powershell
npm run build
```

4. Preview production build locally

```powershell
npm run preview
```

## Available scripts (from package.json)

- `dev` — start Vite dev server
- `build` — build production bundle
- `preview` — locally preview the production build
